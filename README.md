## vote-system
Writen using NestJS and Vue.js.
This application is written for give votes and and to research opinions.

## current version
ALFA 0.0.1 - basic application full work.

## How to run
In first step make sure you have Node.JS and npm.
link to download : https://nodejs.org/en/download

On secound step make sure you have NestJS and Vue.js.
If you haven't open console and use this command:

# For install NestJS
npm i -g @nestjs/cli

# For install Vue.js
npm install -g @vue/cli

# Run api
Go to root directory for api and when you are here for the first time use: npm install.

Before starting, make sure you have completed the file .env.
For example you can use file example.env.

In any case, use: nest start.

# Run web
Go to root directory for api and when you are here for the first time use: npm install.
In any case, use: npm run serve.
